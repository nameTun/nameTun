### 🌟 Featured Project: BandMates AI Platform
*A Modular Monolith backend designed to evaluate IELTS essays and intelligently suggest vocabulary practices via Generative AI.*
- **Clean Architecture:** Applied Global Interceptors and Exception Filters to standardize payloads.
- **Security:** Engineered an XSS-resistant Google OAuth2/JWT flow with HttpOnly Cookies and "Silent Refresh".
- **System Design:** Developed a Hybrid Rate Limiting system (IPv4/IPv6 + VisitorID tracking).
- **Fault Tolerance:** Employed TypeScript Generics and Exponential Backoff strategy to auto-recover from API 503 errors.

**🚧 Roadmap / Currently in Development:**
*   **Redis Integration:** Optimizing system performance and token management through centralized Redis caching.
*   **Custom Prompt Evaluation:** Enabling users to submit their own custom IELTS prompts for AI-powered scoring and detailed feedback.
*   **Advanced Phrase Analytics:** Implementing a lookup feature for high-tier IELTS grammatical structures and academic collocations based on specific input phrases.
